# I18n Rules

## Locale Refetch

- Saat locale/bahasa berubah, refetch data API yang memang dipengaruhi locale pada halaman/komponen aktif.
- Jangan anggap perubahan bahasa hanya mengganti teks statis frontend.
- Jika page memakai data backend yang mengikuti header `Accept-Language`, page/composable tersebut wajib memanggil ulang service relevan.

## Contoh Flow

- Profile aktif perlu refetch `/auth/me`.
- Halaman users perlu refetch list users.
- Halaman detail perlu refetch detail data yang sedang tampil.

## Implementasi

- Utamakan pola refetch dari composable/page yang memiliki data tersebut.
- Gunakan helper/composable reusable jika pola watch locale dipakai di banyak tempat.
